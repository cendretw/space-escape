關於本遊戲設計

設計目的： 結合虛擬互動面板與實體教材翻閱。促使學生在競速的高壓環境下主動閱讀、定位課本、講義（All in One、四合一學習講義）與雜誌（Live互動英語2026年5月號）。

防弊機制： 網頁代碼中的核心答案皆已使用 Base64 雜湊加密。且 UI 面板與 Placeholder 中完全不含有任何字彙提示。學生必須翻書、閱讀，取得語意後方可作答。此外，大關卡之子題目採行 Fisher-Yates 動態洗牌亂序技術，即使並肩而坐其答題順序亦不相同，從根本上杜絕答案互抄與直接通報問題。

部署提示： 將此 index.html 放入全新的 GitHub 儲存庫（Repository），並在 Settings -> Pages 啟用即可線上運行。
